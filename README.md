# KGbing Wallpaper Simple

启动即自动更换今日 Bing 壁纸。支持系统托盘快捷操作，双击切换随机壁纸。

## 功能

- 启动自动更换今日 Bing 壁纸
- 托盘菜单：今日壁纸、随机壁纸、显示主界面、退出
- 双击托盘图标更换随机壁纸
- 定时自动更换壁纸（可配置间隔）
- 画报模式（全屏轮播本地壁纸）
- 空闲自动画报
- 单实例运行（重复打开自动弹出主窗口）

## 运行环境

- Python 3.6+
- Windows

## 快速开始

下载 3.0`，双击运行。

## 使用提示

- 首次运行自动设置今日壁纸并驻留托盘
- 右键托盘图标可切换今日/随机壁纸
- 再次运行 EXE 会弹出已运行实例的主窗口
- 画报模式按 ESC 退出

## 技术栈

Python 3 + PyQt5 + Pillow
